Student Feedback Analyzer using Python and NLP

 Description
This project analyzes student feedback collected through Google Forms
and generates summaries, common issues, wordclouds, and PDF reports.
It helps teachers understand student opinions quickly and improve
teaching quality.

Features
- Upload CSV feedback file
- Automatic text analysis
- Teacher-friendly summary
- Student quote extraction
- Wordcloud generation
- PDF report download
- Web-based interface

Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- WordCloud
- FPDF

How to Use
-Open the application link in any web browser.

-Download the feedback file from Google Forms:

-Go to Responses

-Click Download (.csv)

-Upload the CSV file on the website.

-Wait for the system to process the data.

-View the summarized feedback and visualizations.

-Click Download PDF Report to save the report.
